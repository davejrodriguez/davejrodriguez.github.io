Unity Mobile Controller

davejrodriguez
November 30, 2016

The Unity Mobile Controller is a project I worked on in our Human Computer Interface lab. This experimental remote control project would allow anyone with a smartphone play splitscreen multiplayer in a Unity project.

The control interface was displayed on the smartphone, while the game sans-interface would be displayed on the computer. Upon logging in and connecting, the phone would spawn and link to an avatar. Using multiplayer networking, the phone would send the joystick commands to the PC, where they'd be received and applied. Essentially, the system treated the avatar on the PC and the controller on the phone as two separate players, even though the tandem represented only one player.

This idea was spawned as a solution to visitors' lack of easy interaction with physical installations of our art pieces. With this controller, anyone who downloaded the companion app and entered a pairing code could join the simulation.
